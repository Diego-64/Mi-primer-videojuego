2. Descripción / Concepto del Juego

Lo que tengo pensado es un juego 2D donde el objetivo es sobrevivir a oleadas de enemigos el máximo tiempo posible sin morir.

3. Controles y Mecánicas

Empezando por los controles:

-Las teclas ASDW: Mover al personaje
-La tecla Space: El personaje realiza un pequeño Sprint

El personaje tendra un arma el cual dispara proyectiles a los enemigos

Hay distintos enemigos con estadisticas similares y enemigos elites con mayor tamaño.

4. Tiempo
El objetivo del juego es sobrevivir el máximo tiempo posible.

El tiempo irá aumentando mientras sigas vivo, el tiempo se detendrá si mueres.

5. Visión general
Por ahora, al ser mi primer juego, realizaré todo lo que he puesto por ahora. Ya después le metere más cosas como sistema de combos, jefes, sistema de niveles, etc.
